PokeChirps

A clone of Twitter using a React / Redux front end, a Python / Flask back end, with a PostgreSQL database. Users are able to post chirps, and comment on other users' chirps. Live Link | Repo

Taste Buds

A clone of Meetup.com using a React / Redux front end, a Python / Flask back end, with a PostgreSQL database. Users are able to create and join groups, create and join events, search for events, and filter all attending events by date. Live Link | Repo

Evernote

A clone of Evernote using a React / Redux front end, an Express / Sequelize back end, with a PostgreSQL database. Users are able to create and edit notes, and group those notes by creating and editing notebooks. Live Link | Repo
